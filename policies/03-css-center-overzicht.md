## Wat is het CSS Center?

Het CSS Center is het Google-platform waarmee CSS-operators hun merchant-portfolio beheren.

**Hoofdfuncties:**
- Overzicht van alle beheerde merchant-accounts
- Geaggregeerde performancedata over alle accounts
- Beheer van gebruikerstoegang en domeinen
- Account diagnostiek, itemstatus en verbetermogelijkheden
- Overdracht van merchant-accounts tussen CSS-providers
- Labelbeheer voor merchants

**Beschikbare rapporten:**
| Rapport | Doel |
|---------|------|
| CSS requirements | Compliancecheck op programma-eisen |
| Account diagnostics | Problemen op accountniveau identificeren |
| Items diagnostics | Productstatus per item monitoren |
| Opportunities | Verbeterpunten in het portfolio |

## Werken met Comparison Shopping Services (voor merchants)

**Bron:** https://support.google.com/merchants/answer/12652686

### Servicemodellen
- **Managed**: CSS maakt Merchant Center en Google Ads account aan, beheert feeds en campagnes.
- **Self-service**: CSS maakt accounts aan en geeft merchant toegang; merchant beheert zelf.
- **Hybrid**: combinatie van bovenstaande.

### Geografische restricties
- Meerdere CSS-providers zijn toegestaan in CSS-programmalanden.
- Slechts **1 CSS per account** voor gratis listings in Europa.
- Slechts 1 CSS voor plaatsingen buiten Europa.

### Switching
Een Merchant Center-account kan maar aan één CSS tegelijk worden gekoppeld.
